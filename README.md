# add_to_start
Shell-script add shorcut to start menu. Скрипт для создания ярлыка в меню "Пуск".

**Инструкция**: 
Замените значение $AppPath = "D:\свой_путь_к_приложению". 
Образец: $AppPath = "D:\Exemple\Exemple\Exemple.exe" Сделай хороший перевод на английский
Для запуска используйте ПКМ-меню и выбирите пункт: Выполнить с помощью PowerShell.

**Instruction**:
Replace the value of $AppPath with your application path.
Example: $AppPath = "D:\Example\Example\Example.exe"
To run it, right-click the file and select Run with PowerShell.
